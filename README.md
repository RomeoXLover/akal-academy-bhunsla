# Akal Academy Bhunsla - School Management Portal

A modern, fast, and comprehensive school management application built for Akal Academy Bhunsla. Available as both a responsive web application and a dedicated Android application (APK).

## Features

- **Role-Based Access Control**: Different views and capabilities for Operators, Principals, Teachers, Majors, and Students.
- **Real-Time Cloud Sync**: Built-in Vercel serverless integration ensures tests, homework, and student records sync across all devices instantly.
- **Offline Support**: Fully functional offline with data persistence. Syncs automatically when the connection is restored.
- **Multi-Language Support**: Available in English, Hindi, and Punjabi.
- **Academic Management**:
  - Class Tests & Homework tracking
  - Exam Results & Report Cards
  - Syllabus & Timetable management
  - Student Profiles & Records

## Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (No frameworks for maximum speed)
- **Mobile**: Capacitor (Generates native Android APK from web assets)
- **Backend/API**: Vercel Serverless Functions
- **Database**: GitHub Repository (`akal-data`) via API
- **Design**: Custom UI with fluid responsive scaling, optimized specifically for modern mobile displays.
